# Contributing

Thanks for considering contributing to Vessel.

## What to Work On

- **TODO.md** — Pick any item from the project's TODO list.
- **Open Issues** — Look through the [issues](https://github.com/iamsurjog/Vessel/issues) for something that needs fixing.

If the thing you want to work on doesn't have an open issue yet, open one first. This keeps everyone on the same page and avoids duplicate work.

## Testing Other Python Versions

The project targets Python 3.14. If you test it on other versions (3.12, 3.13, etc.) and things work — or don't — feel free to update the README note accordingly so others know what's been tried.
