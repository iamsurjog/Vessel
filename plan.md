# Vessel — Improvement Ideas

## 🔥 High Impact

1. **Flashcards + Spaced Repetition** — AI generates Q/A pairs from materials; built-in review loop with SM-2 scheduling
2. **Quiz Yourself Mode** — AI generates multiple-choice, short-answer, fill-in-the-blank quizzes; score tracking
3. **Study Plan Generator** — AI creates a day-by-day study schedule from your materials and deadline

## 🧠 Smart Study (Builds on existing RAG)

4. **Cross-Vessel Search** — global search across all vessels, not just the current one
5. **Concept Mapping** — AI identifies key concepts and draws connections between them
6. **Automated Study Guides** — structured printable study guide generated from materials
7. **Writing Feedback** — essay/assignment review with argument, clarity, and grammar analysis

## 🎨 UX & Productivity

8. **Split-Pane Mode** — view two notes or a note + material side-by-side
9. **Quick Capture** — global hotkey / system tray for rapid note entry
10. **Document Annotation** — highlight PDFs and materials; highlights become notes
11. **LaTeX / Math Rendering** — inline math in markdown view for STEM students
12. **Full-Text Search in Droplets** — BM25 search should also index notes, not just materials
13. **Syntax Highlighting** — code block highlighting in the editor

## 🚀 Distribution

14. **One-Click Installer** — package as standalone binary (PyInstaller / Nuitka)
15. **Multi-Device Sync** — export/import vessels; optional cloud or Syncthing integration

## 🧪 Polish

16. **Progress Indicators** — upload progress bar, AI status messages, toast notifications
17. **Onboarding Flow** — first-run tutorial explaining Vessels, Materials, Droplets, AI chat
18. **Undo / Recycle Bin** — soft-delete for accidentally removed files
